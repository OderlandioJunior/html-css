Cascading Style Sheets Inline

O CSS inline é usado para dar estilo a um elemento HTML específico. Para este estilo de CSS, você somente vai precisar adicionar o atributo style para cada tag HTML, sem usar os seletores. Este tipo de CSS não é realmente recomendado, já que cada tag HTML precisa ser estilizada de maneira individual. Para inserir o CSS Inline basta adicionar o parâmentro style="" dentro da tag e preencher com o comando desejado fechando com ";".

Cascading Style Sheets Interno / Local

O CSS interno consiste em adicionar as regras de CSS no mesmo documento em que está o conteúdo da página, logo depois da tag <head>, criando uma tag de <style> e </style> é possivel adicionar SELETORES para as determinadas tags. O seletor da tag por sua vez é configurado dentro de chaves "{}" sendo necessário fazer a declaração com propriedade/atributo seguido de ":" e o valor para configurar o estilo da tag fechando com ";".

Por exemplo esse trecho que está com uma formatação de cor interna!

Cascading Style Sheets Externo

O CSS externo permite incorporar regras de CSS presentes em um documento (.css) à parte a outro arquivo HTML, em que ficam os elementos da página. A ligação dos documentos é feita através da tag <link:css> aberta dentro da tag <head>

Hierarquia das CSS

Existe uma ordem de prioridades nas folhas de estilos na qual a INLINE sempre valerá mais que a EXTERNA, dessa forma, podemos estilizar toda a página e se quisermos modificar em apenas uma tag, basta usar a CSS INLINE.

Seletores CSS

Ao criar uma folha de estilo para o documento html é necessário usar seletores para as devidas marcações, no entanto, ao usar um seletor na folha de estilo, a configuração feita será concedida a todas as outras marcações, como ja vimos anteriormente, então, para diferenciar a mesma marcação das demais, utilizaremos os seletores de id ( # ) e de class ( . ) de maneira eficiente. Ao criar nosso conteúdo em HTML, podemos identificar um determinado elemento único com um id (Pode ter mais de um id, o que nao pode é ter o mesmo id pra elementos diferentes. O nome de um id é exclusivo somente pra aquele elemento e o que não se pode fazer é repetir esse nome pra outro elemento), ou agrupar elementos múltiplos que tenham características semelhantes com um class, dessa forma, criamos seletores personalizados para atribuir um estilo as marcações dentro do documento.

Seletores personalizados

Nas CSS, os identificadores atuam na identificação de uma marcação, representado como #, esse seletor é capaz de aplicar uma ênfase em uma determinada marcação, pois em um único documento, só podemos usar um id para um único elemento, o que significa que não podemos ter dois ou mais elementos com um mesmo id dentro de um mesmo documento html, como por exemplo três "h1" com um id "principal", claro que a estilização feita será concedida aos três elementos, mas como o HTML5 é fundamentado pela semântica, isso estaria errado. As classes, por sua vez, são responsáveis para atribuir configurações específicas a um grupo de elementos

Seletores especiais

Pseudo-classes

Representado através de ( : ) nas CSS, a pseudo-classe altera o estado de um elemento, ou seja, ela muda a forma de acordo com a interação do mouse do usuário com o devido elemento, tendo seu principal uso, o :hover.

Pseudo-elementos

Inserido após usar duas vezes dois pontos ( :: ) o pseudo-elemento altera o conteúdo do elemento através das CSS sem ter que necessáriamente alterá-lo no HTML

Propriedade em CSS

As propriedades são responsáveis para criar ações nas tags, sendo assim, temos as definições de algumas:

Propriedades / Ação color / Altera a cor da letra background / Altera o fundo da página, texto e/ou imagem (Possui valores como: color, image, size, position) border / Cria uma borda em volta do elemento (Possui valores como: color, image, size, position) margin / Cria um espaço em volta do elemento (Possui valores como: orientação, sendo top, bottom, left, right) padding / Cria um espaço dentro do elemento (Possui valores como orientação, sendo top, bottom, left, right) height, width / Altera o tamanho do elemento text / Transforma uma caixa de texto (Possui valores como -align, -decoration, -transform, -indent, -letter-spacing, -word-spacing e -line-height) font / Altera a estrutura do texto (Possui valores como: -family, -size, -style, -weigth) display / Capaz de controlar o layout da página (Possui valores como :block, :inline, :inline-block, :none)
