Construção de textos

Ao desenvolver um site com textos e imagens, precisamos primeiramente entender o conceito da marcação HTML. A atualização da HTML4 para HTML5 trouxe consigo uma mudança significativa que foi o acréscimo da Semântica em suas tags. Esse ramo linguístico traz significados internos para mecanismos de buscas e algoritmos além da própria ação da tag.

Tags semânticas e não semânticas para formatações de texto

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag B (não semântico).

Nesta frase, temos um termo em destaque usando a tag STRONG (semântico).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semântico).

Nesta frase, temos um termo em ênfase usando a tag EM (semântico).

Texto marcado

Podemos fazer também um texto marcado usando a tag MARK.

Texto pequeno

Podemos criar também um texto pequeno usando a tag SMALL.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado usando a tag DEL.

Texto inserido

Podemos marcar um texto como inserido para dar um ênfase e indicar que ele foi adicionado depois, usando a tag INS(semântico).

Existe também o sublinhado com a tag U(não semântico).

Texto sobrescrito

Para inserir coisas do tipo: x20+3, usamos a tag SUP.

Texto subescrito

Para inserir coisas do tipo: H2O, usamos a tag SUB.

Citações simples

A tag Q é usada quando há uma citação simples, por exeplo: Fulano falou O computador é um burro muito rápido

Citações completas

A semântica para usar a citação completa ocorre quando é uma citações de algum autor ou algo do tipo, dessa forma usamos a tag BLOCKQUOTE para fazer a citação do autor.

João 14: 5 Disse-lhe Tomé: "Senhor, não sabemos para onde vais; como então podemos saber o caminho?" João 14: 6 Respondeu Jesus: "Eu sou o caminho, a verdade e a vida. Ninguém vem ao Pai, a não ser por mim.

Abreviações

Podemos mostrar o significado de tal abreviação usando a tag ABBR: HTML e CSS

Outras formatações

Código-fonte

Para mostrar na pagina o comando que foi feito, usamos a tag CODE, dessa forma, na exibição, a formatação da letra terá o mesmo tamanho semântico de comandos, por exemplo: O comando document.getElementById("teste") é escrito em linguagem JAVASCRIPT.

Pré-textos

Quando usamos a tag PRE, tudo que está dentro dessa tag será exibido na pagina, ou seja, a tag PRE considera as quebras de linhas usando o ENTER e as tabulações. Normalmente usadas para exibir totalmente as linhas de códigos, como por exemplo, um codigo em linguagem PYTON.

Hierarquia de títulos

Assim como a semântica trouxe significados internos para algumas tags, ela também trouxe uma hierarquia de títulos, aonde temos até o nível 6 de relevância. Ou seja, seguindo uma lógica onde o <H1> apresenta o título principal do conteúdo e por isso tem maior relevância até o <H6> que pela lógica tem menor importância.
