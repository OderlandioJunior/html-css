Como fazer uma ancoragem de um arquivo

Usa-se o termo "âncora" para relacionar a sua tag <a> e não confundir com a tag <link>. Essa tag tem como objetivo tornar um texto ou até imagens clicável e direcionar o usuário para outro conteúdo. Para criar um link em determinado objeto usamos o seguinte comando <a href="">

Trabalhando com links EXTERNOS

Em situações que é necessário criar um link que leve o usuário para outro conteúdo é necessário fazer algumas modificações no parâmetro da tag <a>

Tag padrão

Seguindo o conceito padrão da tag <a href=""> temos o seguinte exemplo: Vamos abrir o repositorio do professor clicando aqui

Seguindo esse conceito, o site ancorado será sobreposto pelo link e o usuário será forçado a voltar a sua página.

Adicionando parâmetros

Com a tag <a href=""> aberta podemos adicionar mais um parâmetro sendo eles o "target=" e o "rel=".

O parâmetro "target=" e o "rel=" tem mais atributos que podem ser vistos no repositorio do professor, mas por se tratar de um site externo, podemos apenas colocar o "_blank" dentro do parâmetro "target=" e "external" dentro do "rel=". Com esse atributo, o navegador do usuário irá abrir uma guia a mais deixando duas abas abertas, sendo a da página principal e a segunda sendo ancorada

Dessa forma, atualizando a tag com os parâmetros adicionais e clicando aqui iremos para o canal do YouTube e podemos ver o resultado da atributo adicional.

Trabalhando com links INTERNOS

Para indexar uma página ou arquivo interno, a tag permanece no seu conceito padrão: <a href="">. Assim, podemos abrir novas páginas dentro do próprio site, contanto que a tag esteja preenchida com o caminho de arquivo no parâmetro <href="">.

Podemos embelezar ainda mais nosso código html para os mecanismos de buscas completando o parâmetro "rel=next" para a próxima página e "rel=prev" para a página anterior e "target=_self".

Trabalhando com links de download

Em casos que o usuário tenha que baixar um conteúdo, temos que configurar mais 2 parâmetros, além do "href=", dentro da tag <a >.

Com a tag <a href=""> aberta, digite "download=nomedoarquivo.tipodoarquivo" e o "type=ação.arq". No caso do "type=", temos que completar o parâmetro com o tipo de arquivo corretamente, no site iana.org tem uma lista com todos os media-types.
