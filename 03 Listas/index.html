Listas em HTML

A criação de lista é importante por que elas facilitam a leitura e organizam o conteúdo. Por conta disso, a linguagem HTML disponibiliza vários tipos de lista para produzirmos conteúdo e deixá-lo mais claro e eficiente.

Para a criação de listas em HTML, temos: <ol> e <ul>. Sendo que para cada <ol> e <ul> temos que por um <li> dentro da tag inicial.

Listas ordenadas

A tag <ol> abre uma lista ordenada aonde podemos contabilizar os itens de dentro dessa lista. A contagem pode variar de acordo com algarismo dentro do parâmetro "type=" existem 5 variações, sendo elas: 1 A a I i.

Dessa forma, a tag completa é escrita dessa maneira <ol type=""></ol>
  1. Item numérico "1"
  1. Item alfabeto maiúsculo "A"
  1. Item alfabeto minúsculo "b"
  1. Item romano maiúsculo "I"
  1. Item romano minúsculo "i"

Listas não ordenadas

Nesse exemplo, a tag <ul> abre uma lista onde os itens são apenas marcados. A marcação, pode variar dependendo do estilo escolhido dentro do parâmetro "type=" sendo elas: disc circle square.

Dessa forma, a tag completa é escrita dessa maneira <ul type=""></ul>
  • Item exemplo "disc"
  • Item exemplo "circle"
  • Item exemplo "square"

Termo de definição

Temos a tag <dl> que cria uma lista com termos e definições. Cada termo é escritos entre um <dt> e </dt>, assim como cada descrição de termo é escrita entre <dd> e </dd>. Com isso temos os exemplos para esse tipo de lista:

HTML
Linguagem de marcação para criação de um site.
CSS
Linguagem de marcação para criação de design de um site.
JavaScript
Linguagem de programação para criação de interatividade de um site
